# Python-Diwali-Sales-Analysis
This project analyzes Diwali sales data using Python to understand customer purchasing behavior and identify key trends. The goal is to extract meaningful insights from raw data and present them visually to support better business decisions and improve customer experience.
The primary goal of this project is to extract meaningful insights from raw data and present them in a visually understandable format for business decision-making.


Objectives

• Clean and preprocess the dataset

• Perform exploratory data analysis (EDA)

• Visualize sales trends and customer behavior

• Identify top-performing segments and product categories

• Provide data-driven business insights


Tools & Technologies Used

• Python

• Pandas

• NumPy

• Matplotlib

• Seaborn

• VS Code


Key Insights

• Female customers contributed more to total sales than male customers

• The 26–35 age group was the highest spending segment


Top-performing states:

→ Uttar Pradesh

→ Maharashtra

→ Karnataka


Most purchased product categories:

→ Food

→ Clothing & Apparel

→ Electronics & Gadgets


These insights can help businesses focus on the most profitable customer segments and improve marketing strategies during festive seasons.


Visualizations Included

• Gender-wise sales distribution

• Age group vs total spending

• State-wise sales analysis

• Product category analysis


(Multiple bar charts and count plots were created using Matplotlib & Seaborn.)
